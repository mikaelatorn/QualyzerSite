Requirements

To use Qualyzer, you must have a Java Runtime Environment (JRE) 11 installed. Qualyzer can be installed on the following operating systems (only 64-bit versions):

  • Windows 10
  • Linux
  • Mac OS X

Download

To download the latest version of Qualyzer, click the link that corresponds to your operating system in the list below. Save the file somewhere on your computer (for example, your Desktop, or a download folder).

Downloads coming soon.

Alternately, previous versions of Qualyzer for older operating systems can be found at SourceForge. You can go directly to the Download page and pick the version you want. For example, select QualyzerSetup_1.2.1.msi to install Qualyzer on Windows.

Installing

On Windows, double-click on the installation file, QualyzerInstaller-2.5.0.exe and follow the instructions on the screen. A shortcut to Qualyzer will be created in your start menu and on your desktop at the end of the installation. Just double-click on the shortcut to start Qualyzer.

Due to security issues regarding downloading unsigned software on Mac OS X, a specific installation procedure has to be followed. A Readme file is provided with the application containing instructions on how to install the program and use it like a normal application.

Alternatively, if you downloaded an archive file (zip or tar.gz), unzip the file in a directory of your choice (the winzip program can unzip files on Windows). Then, double-click on the file qualyzer.exe in the qualyzer folder (or just the qualyzer file on a Mac or Linux operating system).
