Requirements

To use Qualyzer, you must have a Java Runtime Environment (JRE) 6 installed. Qualyzer can be installed on the following operating systems:

  • Windows XP, Vista, 7 (32 and 64 bit editions)
  • Linux (32 bit and 64 bit editions)
  • Mac OSX (32 bit and 64 bit editions)

Download

To download the latest version of Qualyzer, click the link that corresponds to your operating system in the list below. This will take you to the SourceForge download website and open a pop-up window asking you to save a file. Click "Save File" and save the file somewhere on your computer (for example, your Desktop, or a download folder).

  • Windows
  • Mac (32 bit)
  • Mac (64 bit)
  • Linux (32 bit)
  • Linux (64 bit)

Alternately, you can go directly to the Download page and pick the version you want. For example, select QualyzerSetup_1.2.1.msi to install Qualyzer on Windows.

For the Mac or Linux operating system, download the version of Qualyzer that matches your JRE edition, e.g., qualyzer_1.2.1_mac32bit.tar.gz for the JRE 32 bit and qualyzer_1.2.1_mac64bit.tar.gz for the JRE 64 bit.

Installing

On Windows, double-click on the installation file, QualyzerSetup_1.2.1.msi and follow the instructions on the screen. A shortcut to Qualyzer will be created in your start menu and on your desktop at the end of the installation. Just double-click on the shortcut to start Qualyzer.

Alternatively, if you downloaded an archive file (zip or tar.gz), unzip the file in a directory of your choice (the winzip program can unzip files on Windows). Then, double-click on the file qualyzer.exe in the qualyzer folder (or just the qualyzer file on a Mac or Linux operating system).
